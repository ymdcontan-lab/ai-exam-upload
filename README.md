# AI录入试卷系统

这是一个AI辅助录入试卷的交互式原型页面。

## 功能特点

- 📄 支持上传试题和答案PDF文件
- 📝 填写试卷基本信息（标题、学段学科、来源）
- 🤖 模拟AI识别流程
- 💡 智能表单验证，所有项填写完成后才能提交
- 📱 响应式设计，支持移动端和桌面端

## 在线访问

部署后可直接通过浏览器访问。

## 技术栈

- 纯HTML + CSS + JavaScript
- 无需后端服务器
- 无第三方依赖

## 本地预览

直接在浏览器中打开 `index.html` 文件即可预览。
